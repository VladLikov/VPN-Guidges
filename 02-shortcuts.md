Shortcuts — это приложение Apple для быстрых команд и автоматизаций. Через него можно запускать действия **ВПН — и точка** без ручного открытия приложения.

## Как создать быструю команду

1. Откройте приложение **Shortcuts**.
2. Перейдите на вкладку **Shortcuts**.
3. Нажмите **+** в правом верхнем углу.
4. Нажмите **Add Action**.
5. В поиске введите **ВПН — и точка**.
6. Выберите действие для подключения VPN.
7. Нажмите на название команды сверху.
8. Задайте понятное имя, например **Включить VPN**.
9. Нажмите **Done**.

Теперь команду можно запускать из приложения Shortcuts, через Siri, с виджета или с Action Button.

## Если действия приложения нет в поиске

Попробуйте сделать так:

1. Откройте **ВПН — и точка**.
2. Проверьте, что VPN-профиль уже настроен внутри приложения.
3. Вернитесь в **Shortcuts**.
4. Снова нажмите **Add Action** и найдите **ВПН — и точка**.

Если действие все равно не появляется, значит текущая версия приложения не добавила поддержку Shortcuts. В этом случае можно использовать только те способы запуска, которые доступны внутри самого приложения и iOS.

## Как запускать команду через Siri

1. Откройте **Shortcuts**.
2. Найдите команду **Включить VPN**.
3. Запомните ее название.
4. Скажите: **Hey Siri, Включить VPN**.

Siri запускает команду по ее названию, поэтому лучше использовать короткое и понятное имя.

## Как добавить команду на главный экран

1. Откройте **Shortcuts**.
2. Нажмите и удерживайте команду **Включить VPN**.
3. Выберите **Details**.
4. Нажмите **Add to Home Screen**.
5. При необходимости измените название и иконку.
6. Нажмите **Add**.

После этого команда появится на Home Screen как обычная иконка.

## Рекомендуемые команды

Можно создать несколько отдельных команд:

- **Включить VPN** — подключает VPN.
- **Выключить VPN** — отключает VPN.
- **Переключить VPN** — меняет состояние VPN, если приложение поддерживает такое действие.

Если приложение поддерживает только одно действие, используйте его как основную команду запуска.
