# Как настроить VPN на Action Button

Action Button — это кнопка действия на поддерживаемых моделях iPhone. Ее можно настроить так, чтобы она запускала команду для **ВПН — и точка**.

## Шаг 1. Создайте команду в Shortcuts

1. Откройте **Shortcuts**.
2. Нажмите **+**.
3. Нажмите **Add Action**.
4. Найдите **ВПН — и точка**.
5. Выберите действие для подключения VPN.
6. Назовите команду **Включить VPN**.
7. Нажмите **Done**.

Если у вас уже есть готовая команда для VPN, этот шаг можно пропустить.

## Шаг 2. Назначьте команду на Action Button

1. Откройте **Settings**.
2. Перейдите в **Action Button**.
3. Смахивайте варианты, пока не выберете **Shortcut**.
4. Нажмите **Choose a Shortcut**.
5. Выберите команду **Включить VPN**.

После этого удержание Action Button будет запускать выбранную команду.

## Как пользоваться

1. Нажмите и удерживайте Action Button.
2. Дождитесь запуска команды.
3. Проверьте статус VPN в **ВПН — и точка** или в строке состояния iPhone.

## Если команда не запускается

Проверьте:

1. Команда есть в приложении **Shortcuts**.
2. Команда запускается вручную из Shortcuts.
3. В настройках Action Button выбран режим **Shortcut**.
4. В качестве команды выбрана именно команда для **ВПН — и точка**.
5. У приложения есть разрешения, необходимые для подключения VPN.

Если iPhone не поддерживает Action Button, используйте виджет, иконку команды на Home Screen или запуск через Siri.
