# Автовключение VPN для конкретных приложений

В Shortcuts можно создать автоматизацию: когда открывается выбранное приложение, iPhone запускает команду для **ВПН — и точка**.

Это удобно, если VPN нужен только для отдельных приложений.

## Шаг 1. Создайте команду включения VPN

1. Откройте **Shortcuts**.
2. Нажмите **+**.
3. Нажмите **Add Action**.
4. Найдите **ВПН — и точка**.
5. Выберите действие для подключения VPN.
6. Назовите команду **Включить VPN**.
7. Нажмите **Done**.

Перед созданием автоматизации проверьте, что команда запускается вручную.

## Шаг 2. Создайте автоматизацию при открытии приложения

1. Откройте **Shortcuts**.
2. Перейдите на вкладку **Automation**.
3. Нажмите **+**.
4. Выберите **App**.
5. Нажмите **Choose**.
6. Выберите приложения, для которых нужен VPN.
7. Оставьте включенным вариант **Is Opened**.
8. Нажмите **Next**.
9. Выберите **Run Shortcut**.
10. Выберите команду **Включить VPN**.
11. Отключите **Ask Before Running**, если такой переключатель доступен.
12. Нажмите **Done**.

Теперь при открытии выбранных приложений iPhone будет запускать команду включения VPN.

## Шаг 3. Автоотключение VPN при закрытии приложения

Если нужно отключать VPN после выхода из приложения, создайте вторую автоматизацию.

1. Откройте **Shortcuts**.
2. Перейдите в **Automation**.
3. Нажмите **+**.
4. Выберите **App**.
5. Выберите те же приложения.
6. Включите вариант **Is Closed**.
7. Нажмите **Next**.
8. Выберите команду отключения VPN.
9. Отключите **Ask Before Running**, если такой переключатель доступен.
10. Нажмите **Done**.

Если отдельной команды отключения VPN нет, этот шаг можно пропустить.

## Важные ограничения

- Автоматизация срабатывает при открытии приложения, а не заранее.
- iOS может показывать уведомление о запуске автоматизации.
- Некоторые действия могут требовать подтверждения, если iOS считает их чувствительными.
- Если **ВПН — и точка** не поддерживает Shortcuts, автозапуск через этот способ может быть недоступен.

## Что проверить, если VPN не включается

1. Команда **Включить VPN** работает вручную.
2. В автоматизации выбрано правильное приложение.
3. В автоматизации выбран триггер **Is Opened**.
4. В автоматизации выбрана правильная команда.
5. В приложении **ВПН — и точка** уже настроен VPN-профиль.
6. У iPhone есть интернет-соединение.
