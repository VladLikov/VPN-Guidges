# VPN-Guidges

Markdown-инструкции для приложения **ВПН — и точка**.

## Файлы

- `manifest.json` — список инструкций для приложения.
- `01-widgets.md` — виджеты на Home Screen, Lock Screen и Control Center.
- `02-shortcuts.md` — настройка Shortcuts.
- `03-action-button-vpn.md` — запуск VPN через Action Button.
- `04-auto-vpn-apps.md` — автовключение VPN для выбранных приложений.

## Raw-ссылка на manifest

После загрузки файлов в GitHub приложение может читать manifest по ссылке:

```text
https://raw.githubusercontent.com/VladLikov/VPN-Guidges/main/manifest.json
```
